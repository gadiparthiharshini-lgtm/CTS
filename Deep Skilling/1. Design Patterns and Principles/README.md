# 1. Design Patterns and Principles

Hands-on solutions for the **Design Patterns and Principles** module of the
Cognizant *Digital Nurture — .NET FSE Deepskilling* program.

> **Note:** All solutions are implemented in **C# (.NET 8)** as runnable console
> applications.

## Exercises

| # | Exercise | Pattern | Description |
|---|----------|---------|-------------|
| 1 | [Exercise1-SingletonPattern](./Exercise1-SingletonPattern) | Singleton | A thread-safe `Logger` with a single shared instance across the whole application. |
| 2 | [Exercise2-FactoryMethodPattern](./Exercise2-FactoryMethodPattern) | Factory Method | A document management system that creates Word, PDF, and Excel documents through factories. |

Each exercise folder contains its own `README.md` with the scenario, a concept
explanation, project structure, run instructions, expected output, and key
takeaways.

## How to Run Any Exercise

Requires the **.NET SDK (8.0 or later)**.

```bash
cd <exercise-folder>
dotnet run
```
