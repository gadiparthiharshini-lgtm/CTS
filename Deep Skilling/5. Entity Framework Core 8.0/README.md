# 5. Entity Framework Core 8.0 — Hands-On Solutions

EF Core 8.0 guided hands-on labs for the Cognizant **Digital Nurture — .NET FSE
Deepskilling** program. A single **.NET 8** console app models a retail store's
inventory in **SQL Server** using **Entity Framework Core 8.0**.

## Index

| # | Hands-on | Labs | Description |
|---|----------|------|-------------|
| 1 | [EF Core 8.0 HOL](./EF%20Core%208.0%20HOL/) | 1–7 | Retail Inventory System — ORM concepts, `DbContext`, CLI migrations, and CRUD + LINQ over `Product`/`Category`. |

**Labs 1–5** are mandatory; **Labs 6–7** are the additional important hands-on.

## Tooling

- Target framework: `net8.0`
- `Microsoft.EntityFrameworkCore.SqlServer` `8.0.8`
- `Microsoft.EntityFrameworkCore.Design` `8.0.8`
- `dotnet-ef` CLI for migrations

See [EF Core 8.0 HOL/README.md](./EF%20Core%208.0%20HOL/README.md) for the full
lab-by-lab walkthrough, run instructions, and expected output.
