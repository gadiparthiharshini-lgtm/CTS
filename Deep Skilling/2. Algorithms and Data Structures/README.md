# 2. Algorithms and Data Structures — Hands-On Solutions

Hands-on exercises for the Cognizant **Digital Nurture - .NET FSE Deepskilling**
program. All solutions are implemented in **C# (.NET 8)** as runnable console apps.

## Exercises

| # | Exercise | Description |
|---|----------|-------------|
| 2 | [Exercise2-EcommerceSearch](./Exercise2-EcommerceSearch) | E-commerce product search comparing **linear search** O(n) vs **binary search** O(log n), with Big O analysis. |
| 7 | [Exercise7-FinancialForecasting](./Exercise7-FinancialForecasting) | Recursive **financial forecasting** (future value of an investment), plus optimization via iteration / memoization. |

## Running an exercise
```bash
cd Exercise2-EcommerceSearch   # or Exercise7-FinancialForecasting
dotnet run
```
Requires the **.NET 8 SDK**.
